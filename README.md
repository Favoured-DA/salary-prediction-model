# Salary Prediction Model

Random Forest Regressor for predicting monthly salary based on employee features (TechCrush AI Bootcamp - Phase 2).

## Features
- Data cleaning & imputation
- Ordinal + one-hot encoding
- Model training & evaluation (R², MAE, MSE)
